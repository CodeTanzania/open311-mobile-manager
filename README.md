open311-mobile-manager
======================

[open311-api](https://github.com/CodeTanzania/open311-api) mobile UI for managers.


## Setup
- Ensure legacy ionic view
```sh
$ ionic config set -g backend pro
```

## Quick Fixes
```sh
$ rm -rf node_module
$ npm install
$ npm rebuild node-sass
```
